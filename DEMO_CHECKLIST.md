# Demo Sanity Checklist

Before showing the demo:

- npm run build passes
- app opens on mobile
- start screen works
- first tap unlocks audio
- language select works
- Mandarin path enters the cat story demo
- preview image carousel works
- preview audio/vocab track works if available
- primer opens and can play vocabulary audio
- story screen opens
- story audio plays or fails gracefully
- recall screen works enough to continue
- reflection/finish screen works
- no console errors
- no broken image icons
- no missing critical audio in the main Mandarin demo path
