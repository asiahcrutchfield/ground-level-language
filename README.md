## English

### Ground Level Languages

There is no shortage of language learning resources.

Courses. Apps. Books. Games.

Most learners try several of them.

And still struggle.

Not because they aren’t trying hard enough—but because something more fundamental is missing.


### The Problem

Most language learning starts too high.

It assumes you can already:
- hear sounds clearly  
- separate words in real speech  
- map sounds to meaning  

But for many learners, this isn’t true.

So they end up:
- memorizing words they can’t recognize when spoken  
- studying grammar they can’t hear in real time  
- consuming input that isn’t actually comprehensible  

The result:

You “know” the language…

…but you can’t hear it.


### What’s Missing

The lowest level.

Before vocabulary.  
Before grammar.  
Before sentences.

There is:
- sound  
- rhythm  
- timing  
- perception  

This is the level children build first.

Adult learners usually skip it.


### The Idea

Ground Level Languages starts at the root of spoken language and rebuilds from there.

Instead of starting with abstract concepts, it starts with:
- simple, concrete experiences  
- repeatable sound patterns  
- highly controlled input  

Language is introduced the way perception develops:
- from sound → to meaning  
- from simple → to complex  
- from concrete → to abstract  


### Current Focus

1. Ear training (phonemes, rhythm, tone)  
2. Vocabulary acquisition without translation  
3. Ultra-simple story input (comprehensible input)  
4. Native-speed speech

## 中文（繁體）

（不翻譯，直接理解）

### 基礎語言學習

語言學習的資源很多。

課程、App、書、遊戲。

很多人都試過。

但還是覺得難。

不是因為不夠努力，而是因為少了一個更基礎的東西。


### 問題

多數語言學習一開始就太難。

它假設你已經可以：
- 清楚聽到聲音  
- 在真實語音中分辨單字  
- 把聲音對應到意思  

但很多人其實做不到。

所以最後變成：
- 記得單字，但聽不出來  
- 學了文法，但聽不懂  
- 看很多內容，但其實不懂  

結果是：

你好像「知道」這個語言…

但你聽不懂。


### 缺少的是什麼

最底層。

在單字之前。  
在文法之前。  
在句子之前。

有：
- 聲音  
- 節奏  
- 時間感  
- 感知  

小孩就是先從這裡開始。

大人通常直接跳過。


### 核心想法

Ground Level Languages 從語言最根本開始建立。

不從抽象概念開始，而是從：
- 簡單、具體的經驗  
- 可重複的聲音模式  
- 高控制的輸入  

語言的引入方式跟感知發展一樣：
- 從聲音 → 到理解  
- 從簡單 → 到複雜  
- 從具體 → 到抽象  


### 目前重點

1. 聽力訓練（音素、節奏、聲調）  
2. 不透過翻譯學習單字  
3. 超簡單故事輸入（可理解輸入）  
4. 母語速度語音
